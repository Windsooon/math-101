##### 基本知识
- 任何不为 0 的数 x 的 -1 次方等于它的倒数
- 1 的任何次幂都为 1
- 0 的正数幂都等于 0。0 的负数幂没有定义。
- 任何非 0 之数的 0 次方都是 1
- -1 的奇数幂等于 -1
- -1 的偶数幂等于 1

## 统计学
#### 概率

|  符号     | 例子       | 含义     |
|  :----:   | :----:     | :----:   |
|  P(A)     | P(A) = 0.5 | A 事件发生的概率 |
|  ∩        | P(A ∩ B) = 0.5 | A，B 事件同时发生的概率 |
|  ∪        | P(A ∪ B) = 0.5 | A 或 B 事件发生的概率 |
|  \|       | P(A | B) = 0.5 | B 事件发生的情况下，A 事件发生的概率 |
|  E        | E(A) = 2   | A 事件的期望值 |
|  var(X)   | var(X) = 2 | X 变量的方差 |
|  σ        | σ = 2      | X 变量的标准差 |
| N(μ,σ²)   | X ~ N(0,3) | 正态分布 |
| U(a,b)    | X ~ U(0,3) | 均匀分布 |

#### 集合
|  符号     | 例子         | 含义     |
|  :----:   | :----:       | :----:   |
|  { }      | {1, 2, 3, 4} | 集合     |
|  A ∩ B    | A ∩ B = {1, 2} | 属于 A 也属于 B 的元素|
|  A ∪ B    | A ∪ B = {1, 2, 3} | 属于 A 或属于 B 的元素 |
|  A ⊆ B    | {1, 2, 3} ⊆ {1, 2, 3} | A 小于或等于 B |
|  A ⊂ B    | {1, 2} ⊆ {1, 2, 3} | A 小于 B |
|  A ⊄ B    | {1, 4} ⊄ {1, 2, 3} | A 不小于 B |
|  P(A)     | P({1,2}) = { {}, {1}, {2}, {1,2} } | A 全部子集的集合 |
|  A - B    | {3,9,14} - {1,2,3} = {9,14} | 从 A 集合减去 B 集合的元素 |
|   Ø       | Ø={}         | 空集合 |
|   U       | U = {1, 2, 3, 4} | 全部可能值的集合 |
|   ℕ₀      | ℕ₀ = {0, 1, 2, 3, ...} | 自然数的集合 | 
|   ℕ₁      | ℕ₁ = {1, 2, 3, 4, ...} | 正整数的集合 |
|   ℚ       | ℚ = {x| x = a/b, a∈ℕ₀, b∈ℕ₁ | 有理数的集合 |
|   ℝ       | ℝ = {x| -∞ < x <∞} | 实数的集合 |
