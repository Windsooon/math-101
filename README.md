### 一元运算符

|  符号     | 例子     | 含义       |
|  :----:   | :----:   | :----:     |
| -         | -1       | 1 的相反数 |
| ++        | ++1, 1++ | 1 自增     |
| --        | --1, 1-- | 1 自减     |
| !         | ! 1      | 非 1       |
| \|\|      | \|1\|    | 1 的绝对值 |
| ^         | ![ex](./imgs/ex/ex.svg)      | 2 * 2 * 2 * 2 |
| √         | ![squrt](./imgs/ex/square_x.svg) | 2 的平方根 |
| ∜         | ![squrt_n](./imgs/ex/square_n_x.svg) | 2 的 4次 方根 |
